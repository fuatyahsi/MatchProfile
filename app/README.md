# MatchProfile App

Flutter client for MatchProfile.

## Includes

- welcome and onboarding flow
- dashboard
- reflection composer
- mock `insight_report_v1` rendering
- validation step
- journal and settings

## Run

```powershell
C:\flutter\bin\flutter.bat pub get
C:\flutter\bin\flutter.bat analyze
C:\flutter\bin\flutter.bat test -r expanded
```
