# Store Release Checklist

## Product Identity

- [ ] Final app name lock
- [ ] Bundle identifier / applicationId lock
- [ ] App icon set
- [ ] Splash / launch assets
- [ ] Marketing subtitle and short description

## Legal & Policy

- [ ] Privacy Policy URL
- [ ] Terms of Use URL
- [ ] KVKK beta policy text
- [ ] Account deletion instructions
- [ ] Data retention explanation
- [ ] Age gate wording for 18+

## App Store Connect

- [ ] Apple Developer team configured
- [ ] App record created
- [ ] Bundle ID created
- [ ] Signing / certificates checked
- [ ] App Privacy answers completed
- [ ] Age rating completed
- [ ] Screenshots uploaded
- [ ] App review notes written

## Google Play Console

- [ ] App created in Play Console
- [ ] Package name locked
- [ ] Signing config verified
- [ ] Data safety form completed
- [ ] Content rating completed
- [ ] Store listing copy uploaded
- [ ] Screenshots / feature graphic uploaded
- [ ] Test track created

## Build & QA

- [ ] `flutter analyze`
- [ ] `flutter test -r expanded`
- [ ] Android debug smoke test
- [ ] Android release build smoke
- [ ] iOS simulator smoke test
- [ ] push notification permission copy QA
- [ ] onboarding copy QA
- [ ] safety escalation copy QA

## Beta Operations

- [ ] First 20–30 user beta list
- [ ] Feedback collection channel
- [ ] Support inbox / issue triage plan
- [ ] Manual review process for first 30 sessions
- [ ] PDV and dispute metrics instrumentation plan
